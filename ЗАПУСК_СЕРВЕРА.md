# Инструкция по запуску сервера

## Вариант 1: Использовать сервер из Downloads (рекомендуется)

Сервер находится в `/Users/ovaam/Downloads/bomberman/`. Запустите его оттуда:

```bash
cd "/Users/ovaam/Downloads/bomberman"
source "/Users/ovaam/Desktop/3 курс/Bomberman/venv/bin/activate"
python server/main.py
```

## Вариант 2: Скопировать сервер в проект

Если хотите, чтобы сервер был в проекте, скопируйте папки:

```bash
cd "/Users/ovaam/Desktop/3 курс/Bomberman/Bomberman"
cp -r "/Users/ovaam/Downloads/bomberman/server" .
cp -r "/Users/ovaam/Downloads/bomberman/maps" .
```

Затем запустите:

```bash
cd "/Users/ovaam/Desktop/3 курс/Bomberman/Bomberman"
source ../venv/bin/activate
python server/main.py
```

## Что должно появиться после запуска:

```
Карта 'arena' успешно загружена.
Карта 'corridors' успешно загружена.
Карта 'maze' успешно загружена.

```

После этого можно запускать iOS приложение!
